# strace

This project implements a system call tracer (similar to `strace`).

## Requirements
- Allowed editors: vi, vim, emacs
- OS: Ubuntu 14.04 LTS
- Compiler: `gcc 4.8.4` using flags `-Wall -Werror -Wextra -pedantic`
- Code style: Betty compliant
- No global variables allowed
- Maximum of 5 functions per file
- Allowed standards: standard C library
- All header files must have include guards
