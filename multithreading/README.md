# Multithreading

This project focuses on multithreading in C and involves working with images (blurring portions using convolution kernels).

## Requirements
- Files are compiled on Ubuntu 14.04 LTS.
- C programs and functions are compiled with gcc 4.8.4 using the flags `-Wall -Werror -Wextra -pedantic`.
- Code must be Betty style compliant.
- Maximum of 5 functions per file.
- Header file `multithreading.h` with include guards.
- All files should end with a new line.
